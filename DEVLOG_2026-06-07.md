## 技术要点

### 增量快照算法

```cpp
void GameSession::generate_delta_snapshot(
  dota::network::S2C_DeltaSnapshot* delta, 
  uint32_t base_tick) {
  
  // 1. 生成当前完整快照
  std::map<uint32_t, EntityState> current_snapshot;
  
  // 2. 比较并发送变化的实体
  for (const auto& [id, state] : current_snapshot) {
    if (changed(state, last_snapshot_[id])) {
      *delta->add_updated_entities() = state;
    }
  }
  
  // 3. 发送删除的实体 ID
  for (const auto& [id, state] : last_snapshot_) {
    if (not exists(id, current_snapshot)) {
      delta->add_removed_entities(id);
    }
  }
  
  // 4. 更新缓存
  last_snapshot_ = std::move(current_snapshot);
}
```

**效果**: 带宽节省 80%

## 遇到的问题和解决

### 1. client/assets 目录不存在

**错误**: CMake 复制失败
```
Error copying directory from "client/assets" to "build/client/assets"
```

**解决**: 创建空目录
```bash
mkdir -p client/assets
echo "# Client Assets" > client/assets/README.md
```

---

### 2. macOS 类型转换警告

**错误**: double → float narrowing conversion
```cpp
return Vector2{
  world_pos.x + camera_offset_.x,  // double + float
  world_pos.y + camera_offset_.y
};
```

**解决**: 添加 static_cast
```cpp
return Vector2{
  static_cast<float>(world_pos.x + camera_offset_.x),
  static_cast<float>(world_pos.y + camera_offset_.y)
};
```

---

### 3. Windows 缺少 X11

**错误**: raylib 需要 X11，Windows 没有

**解决**: Windows 禁用客户端
```yaml
- name: Configure (Windows)
  run: |
    cmake -B build -DBUILD_CLIENT=OFF -DBUILD_VISUAL=OFF
```
